## 0. 文案铁律(2026-07-04 用户,与视觉同级)
- **不写广告语 / 口号 / 自夸**("宁缺毋滥""只贴够格的""扔纸篓"这类留给 README,产品页面只留**事实**:日期 / 条数 / 更新时刻 / 核验说明)。
- **版块不编号**(无①②③④、无第一版第二版),各版平权,只用板名。
- 允许的功能性说明:「数字均对照原文核验」「拿不准的已标待核实」「今日未入选:水货·跑题·不够格」(闸的事实行为)。

## 2. 字体(全系统字体,零外链,离线可渲/可看)
- 剪报标题 / 正文:**宋体** `'Songti SC','STSong','Noto Serif SC',serif`(标题 900,正文常规,正文两端对齐)。
- 手写批注 / 一切"人手写的":**手札体** `'Hannotate SC','HanziPen SC','Kaiti SC',cursive`
  (批注便利贴 / 日期行 / 来源行 / 操作钮 / 索引标签 / 板数)。⚠️ 手札体是 macOS 字体:PNG 在本机渲不受影响;
  网页在非 Mac 设备会退化到楷体/系统体,接受。
- 序号 / 星数 / Pulsewire 词标:**Georgia**(tabular-nums)。

## 3. 组件语言(剪报本隐喻,两端同源)
- **报头**:手写日期行(红笔波浪线)+ `Pulsewire`(Georgia 700)+ `日报`(宋体 900,**红笔圈一圈**,旁一颗红笔星)。
  角上两条和纸胶带;右侧黄便利贴(图钉 tape)写**事实**:N 张 · 每日 15:00 更新。
- **✂ 虚线裁切线** + **「✂ 今日剪自」信源 ticker**(手写体;App 横滚,PNG 静态一行)。
- **导读撕纸条**(App):上下毛边撕纸,「赶时间?先看这几张 ☞」+ 各板头条,点跳。
- **索引标签导航**(App,sticky):手账彩色索引标签(黄/粉/绿/蓝轮换,微旋转),当前板弹起加粗;
  附「在追」「往期」标签 + 「翻本子找…」检索框。实底 + 虚线缝,不透底。
- **编者按**:大黄便利贴(红图钉 + 右下折角),手写体,标题「编辑今天想说 ——」/PNG 版「编者按 ——」。
- **版头**:板名宋体 900 + **荧光扫黄**(App 进场时扫过)+ 红笔手绘箭头 + 手写「N 张剪报」;版导语手写体「☞ 」起头。
- **剪报卡**(核心):
  - 毛边:clip-path 多点撕边(两种齿形轮换);旋转 ±0.6~1.8°;wrapper `filter:drop-shadow`(clip-path 吃掉 box-shadow)。
  - 顶部和纸胶带;头条另加副胶带 + **红笔星** + 标题下红笔波浪线;个别卡别**回形针**(灰笔 SVG)。
  - 「✂ 剪自 <来源>」手写行 → 宋体 900 标题 → tldr(**荧光划已对账数字**)→ 操作行(手写:▸看编辑批注 / 原文↗ / ♡收进本子)。
  - **批注便利贴 = insight**:黄/粉交替,tape 贴角,手写体「批注:」;App 点开展开,PNG 直接摊开。
  - `needs_review` = 红笔圈「待核实?」(印刷体徽标禁用——是编辑手写的疑问)。
  - **「追 · 第N天」红章**(已剪记忆②,2026-07-05):连续多天在追的事件盖一枚歪斜红印——
    手写体 + 红笔描边圆角框 + 轻旋转(PNG `.trkstamp` 在「✂ 剪自」行内;App `.trkmark` 在操作行,
    点击跳「在追」;速读/中详行内〔追·第N天〕;飞书卡链接行 `追·第N天` 代码标)。只陈述事实,不是装饰。
  - GitHub repo 名 = 虚线框 Georgia 芯片,「✂ 」前缀。
- **摘星页(GitHub 板)**:手写榜单行 —— 红笔序号 + 宋体 900 repo + 红「★」+ Georgia 星数(App 进场跳表)+
  一句话 + 批注贴;行间波浪虚线分隔。
- **在追(App)**:线索剪报卡 —— 线名 + 板名章 + 红「已追 N 天」+「✕ 不追了」(本机隐藏可恢复);
  「现状」黄贴 + 左侧虚线时间轴(日期红,条目可点原文)。
- **收藏本(App)**:右缘粉便签抽屉「❤ 收藏本 N」;条目「✕ 撕下来」;点条目跳回原剪报。存 `localStorage pw.fav`(_uid=板key:id,与旧版兼容)。
- **检索(App)**:覆盖层撕边纸「翻本子找 ——」,字面匹配 `<mark>` 荧光高亮,点击跳回原剪报。
- **落款**:✂ 虚线收边;手写「到这儿就贴完了/数字均对照原文核验」;**纸篓彩蛋**:手绘纸篓 + 三团废纸,
  标注当日三道闸的事实「今日未入选:水货 · 跑题 · 不够格」;一圈淡咖啡渍。
- **空版**:黄便利贴「今天这版从缺——没有过审的条目,不硬凑。」(绝不硬凑,绝不留白无解释)。

## 4. 动效(仅网页 App;PNG 全静态)
- 开场时间线:日期行落 → 报头墨迹显影(blur→清晰)→ **红笔圈当场画出**(SVG stroke-dashoffset)→ 便利贴「啪」拍上(scale 回弹)→ 裁切线/ticker 淡入 → 编者按拍上。
- 各版滚动进场:版名荧光扫过 + 剪报逐张贴入(位移+旋转回正回弹,stagger)+ 该版红笔涂鸦开画。
- 摘星星数跳表(rAF count-up);ticker 无限横滚。
- 悬停:剪报摆正 + 浮起 + 影子加深;批注展开 max-height + ▸ 旋转。
- 全部尊重 `prefers-reduced-motion`(动画/过渡关,涂鸦直接呈现)。

## 5. 数字回源(铁律,视觉层)
- 只渲染 verify 对账后的 `summary_rendered`;`needs_review` 条目红笔圈「待核实?」,不静默当真。
- **荧光笔只划已对账的带单位数字**(`_NUM_HIGHLIGHT_RE` / App `hl()`),绝不展示无来源数字。
